Failed to Start Backend

The Cold Outreach Agent backend could not be started. This usually happens when Docker is not running or Node.js is not installed.

Troubleshooting Steps:

  • Make sure Docker Desktop is installed and running
  • Ensure Node.js is installed on your system
  • Check that no other application is using port 8000
  • Try restarting the application
  • Run docker-start-simple.bat manually to see detailed errors
